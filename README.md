# Проектно-технологическая практика, 2026
Индивидуальное задание (тема): Разработка внешнего отчета «Штатное расписание» с использованием языка запросов и системы компоновки данных (СКД) в 1С

Инструкция по формированию отчета: 

1. Запустите информационную базу 1С:Зарплата и кадры государственного учреждения и войдите под своей учетной записью.

2. Откройте внешний отчет
В верхнем меню выберите Файл → Открыть.
Найдите файл внешнего отчета «ВнешнийОтчет_ШтатноеРасписание.erf».
Выделите файл и нажмите Открыть.

3. Заполните параметры отчета
После открытия отчета откроется форма с параметрами.
Необходимым для заполнения параметром является дата формирования отчета. 
Другие доступные параметры отбора заполняются по желанию. При отсутствии заполненных полей отбора отчет будет сформирован по всем доступным данным.

4. Сформируйте отчет
Нажмите кнопку Сформировать.
